App Info
1 Click the Download Easy Installer button to download install.bat.
2 Double-click (or right-click and select Run as Administrator) on the install.bat file.
3 If a Windows Defender screen appears, click More info and then click Run anyway.
4 Click Yes on the UAC prompt to allow installation of the certificate. The installer window will open automatically, and the **Install** button will be fully enabled!
1 Download our official Security Certificate (eotc_present.cer).
2 Double-click the downloaded eotc_present.cer file and click Install Certificate...
3 Select Local Machine as the Store Location and click Next.
4 Select Place all certificates in the following store, click Browse..., select Trusted Root Certification Authorities, and click OK.
5 Click Next, then click Finish. Once imported, click the **Standard App Installer** button to install.
